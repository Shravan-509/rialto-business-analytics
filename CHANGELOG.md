# Changelog

All notable changes to the Rialto Decision Intelligence Platform will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project follows semantic versioning where practical.

## [1.0.0] - 2026-07-19

### Added

- Streamlit dashboard application with executive, sales, customer, returns, satisfaction, sentiment, predictive, forecasting, AI advisor, export, and About pages.
- Dynamic data pipeline for schema validation, cleaning, feature engineering, and processed dataset export.
- NLP pipeline with VADER, optional BERT, word clouds, n-grams, topic extraction, and fallback behavior.
- Predictive analytics pipeline for return classification, satisfaction regression, and customer segment prediction.
- Forecasting pipeline using Prophet with ARIMA and naive fallbacks.
- Gemini-backed Executive AI Advisor with deterministic fallback summaries.
- Power BI export datasets.
- Automated Markdown and PDF report generation.
- Architecture documentation, final release review, MIT license, and contribution guide.

### Notes

- This is the capstone release candidate intended for academic submission, GitHub portfolio use, and technical interview walkthroughs.
